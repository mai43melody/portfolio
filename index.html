Mai Murakami

大学卒業後、2018年から調味料メーカーで約5年間バックオフィスに従事し、法務・営業事務の部署を経験。
在職中より、SNSやweb、パッケージデザインを活用して商品の魅力を伝えることに関心を持ち、Web業界を志す。
2023年9月より職業訓練校で、デザインツールやWeb制作の基礎を学び、2024年2月修了。

web design

Webサイト、バナーなどの制作ができます。細部まで妥協せず、より良いものを作るためにこだわりを持って取り組んでいます。
見た目の良さだけでなく、情報をどのように伝えるかを考えながら、まとまりや配置にも気を配ります。
課題解決に役に立ち、人の心を動かせるようなデザインを目指しています。

tools: illustrator/photoshop/Figma

cording

HTML/CSSを使用し、デザインを正しく実装します。(レスポンシブ対応/scss記述)
またJavascriptを使用し、サイトに動きを加えることができます。
コード記述は規則性を持たせ、誰が見ても分かりやすいコーディング・フォルダ管理を心掛けています。

tools: Visual Studio Code

student time

大学フィギュアスケート部
全く滑ったこともない状態から練習を始め、卒業までの4年間で、5種のシングルジャンプと、4種類のスピンを習得しました。
「できなかったことができるようになる」
自分や仲間の成長と充実に、楽しさ・嬉しさを感じます。
大会運営や幹部メンバーとして後輩の育成を経験し、個人競技としての成長だけでなく、チームとしての成長サポートにも尽力しました。

外国語学部でイタリア語を専攻し、中部のシエナという街に短期留学しました。中でも、西洋美術（絵画・建築）に興味を持ち、その色彩の美しさや構図、作られた背景に関心があり、今もよく美術展に出かけます。
美術を鑑賞することで、デザイン思考も磨かれると思います。
